### S-04: Файл конфига с правильными правами
```bash

# history.db   должен быть 600 (только владелец)
# config.json  должен быть 600 (только владелец)

# В коде при создании файлов:
os.WriteFile(path, data, 0600)   # ✅ правильно
os.WriteFile(path, data, 0644)   # ❌ неправильно — группа и другие могут читать
os.MkdirAll(dir, 0700)           # ✅ директория только для владельца
```

### S-07: Ключ не попадает в логи
```bash
# Проверка: поиск мест где ключ может быть залогирован

grep -rn "fmt.Print" . --include="*.go" | grep -i "key\|apikey"
grep -rn "fmt.Sprintf" . --include="*.go" | grep "APIKey\|apiKey"
```
